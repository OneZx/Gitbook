
### 7. 插入超链接, 图片

> **超链接**, 使用`[超链接名字](地址)`来插入超链接

```
[百度](https://www.baidu.com/)
```

效果: [百度](https://www.baidu.com/)



> **图片**, 使用`![图片名称](地址)`来插入图片

地址可以是`绝对地址`,也可以是`相对地址`


  ```
  ![demo](/assets/demo.png)
  ```

  效果如下

  ![demo](/assets/demo.png)

### 8.表格

> **表格**

- 最简单版,最前面和后面不需要加`|`,也可以加上
```
  id    |   name   |   score
  ---   |   ---    |   ---
  001   |   Mark   |   90
  002   |   Ford   |   80
  003   |   Alan   |   95
```

id | name | score
--- | --- | ---
001 | Mark | 90
002 | Ford | 80
003 | Alan | 95

- 加上'|',效果一样,第二行的短横线可多可少`-----`

```
|id | name | score|
|--- | --- | ---|
|001 | Mark | 90|
|002 | Ford | 80|
|003 | Alan | 95|
```
|id | name | score|
|--- | --- | ---|
|001 | Mark | 90|
|002 | Ford | 80|
|003 | Alan | 95|




- `左,中,右对齐`,用冒号`:`对齐,冒号写左边`:---`则左对齐,右对齐`---:`,居中`:---:`, `:`和旁边的`|`最好不要有空格

```
| id | name | score |
|:--- |:---:| ---:|
| 001 | Mark | 90 |
| 002 | Ford | 80 |
| 003 | Alan | 95 |

```

| id | name | score |
| :--- | :---: | ---: |
| 001 | Mark | 90 |
| 002 | Ford | 80 |
| 003 | Alan | 95 |
